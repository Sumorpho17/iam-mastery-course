## 🎯 What This Is

A **12-week, 60-day** professional IAM course built as a progressive web app. It covers everything from Active Directory fundamentals to AI Agent identity governance — designed for the 2026 job market.

**Live Course:** [Launch the Course →](https://Sumorpho17.github.io/iam-mastery-course/)

## 📋 Curriculum Overview

| Week | Topic | Key Technologies |
|------|-------|-----------------|
| 1 | IAM Foundations & Active Directory Part I | AD DS, Kerberos, GPO, RBAC/ABAC |
| 2 | AD Advanced & Hybrid Identity | Tier Model, Entra ID, PIM, Azure AD Connect |
| 3 | AWS IAM & Cloud Identity | IAM Roles, SCPs, Federation, OIDC |
| 4 | Federation Protocols Deep Dive | SAML 2.0, OAuth 2.0, OIDC, SCIM 2.0 |
| 5 | Identity Governance & PAM | IGA, Access Reviews, CyberArk, Entitlement Mgmt |
| 6 | Agentic AI & Non-Human Identity | AI Agent Identity, MCP Security, NHI Governance |
| 7 | Zero Trust & Capstone Project | Zero Trust Architecture, Integration Project |
| 8 | GCP IAM & Multi-Cloud Strategy | GCP IAM, Workload Identity, Cross-Cloud Mapping |
| 9 | Kubernetes & Container Identity | K8s RBAC, IRSA, SPIFFE/SPIRE, Service Mesh |
| 10 | Identity Threat Detection (ITDR) | Entra ID Protection, GuardDuty, Forensics |
| 11 | Compliance Frameworks & Audits | SOX, HIPAA, GDPR, PCI-DSS, ISO 27001 |
| 12 | Career Acceleration | Resume, Interviews, SC-300, CISSP, Consulting |

## ✨ Features

- **📱 iPad Optimized** — Touch-friendly, responsive design, installable as a home screen app
- **💾 Progress Tracking** — Mark days complete, progress saves to your browser (localStorage)
- **👆 Swipe Navigation** — Swipe left/right to navigate between days on touch devices
- **🌙 Dark Mode** — Easy on the eyes for long study sessions
- **📖 Four Learning Modes** — Learn (concepts), Lab (hands-on), Resources (links), Quiz (self-check)
- **🔌 Zero Dependencies** — Single HTML file + data file. No framework, no build step, no server needed.

## 🚀 Getting Started

### Option 1: GitHub Pages (Recommended for iPad)
Visit the live site and pin it to your home screen:
1. Open the course URL in Safari on your iPad
2. Tap the Share button → "Add to Home Screen"
3. It launches as a standalone app with your progress saved

### Option 2: Run Locally
```bash
git clone https://github.com/Sumorpho17/iam-mastery-course.git
cd iam-mastery-course
# Open index.html in any browser — that's it!
```

## 🛠 Tech Stack

- **HTML5** — Semantic, accessible markup
- **Vanilla CSS** — Custom properties, responsive design, no framework
- **Vanilla JavaScript** — No React, no build step, just clean JS
- **localStorage** — Progress persistence across sessions
- **PWA** — Installable, works offline after first load

## 📜 Prerequisites

- **CompTIA Security+** (or equivalent security knowledge)
- **Motivation** to specialize in IAM
- A computer for labs (VirtualBox for AD lab, free cloud accounts for cloud labs)
